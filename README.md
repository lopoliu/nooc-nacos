# nooc-nacos
这是一个spring cloud nacos学习项目

[nacos官网地址](https://nacos.io/zh-cn/)

## 安装nacos服务
下载nacos服务: [Github下载](https://github.com/alibaba/nacos/releases)

- 启动nacos  
```sh startup.sh -m standalone```
- 访问nacos控制台  
  - nacos默认端口启动在8848
  - nacos默认账户名*nacos*、默认密码*macos*
```http://localhost:8848/nacos/#```
